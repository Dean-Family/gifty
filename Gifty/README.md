# Gift Giving App

The Gift Giving App is an iPhone application that helps you keep track of gifts for various occasions, when they need to be given, and to whom they will be given.

## Features

- Add gifts with name, recipient, occasion, and due date
- View a list of upcoming gifts and their due dates
- Mark gifts as purchased and/or given
- Edit or delete existing gifts
- Search for gifts by name, recipient, or occasion

## Installation

To install the Gift Giving App, follow these steps:

1. Clone the repository to your local machine
2. Open the project in Xcode
3. Build and run the app on your iPhone or simulator

## Usage

To use the Gift Giving App, follow these steps:

1. Launch the app on your iPhone
2. Add a new gift by tapping the "+" button
3. Fill in the required information, such as the gift name, recipient, occasion, and due date
4. Tap "Save" to add the gift to the list
5. View upcoming gifts by tapping the "Upcoming" tab
6. Mark gifts as purchased and/or given by tapping the "Purchased" or "Given" button
7. Edit or delete existing gifts by swiping left on the gift in the list and tapping "Edit" or "Delete"
8. Search for gifts by tapping the search bar and entering a name, recipient, or occasion
